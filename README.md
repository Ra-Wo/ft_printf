# ft_printf

This is a recoding of the printf function, made as an assignment at [1337][1]. It works with the c, s, p, d, i, u, x, X, %.

# How to run it

```bash
git clone https://github.com/Ra-Wo/ft_printf.git && cd ft_printf && make

```
